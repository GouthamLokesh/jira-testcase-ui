UI Test Scenarios

Positive Scenarios

PS001 - Generate Test Cases
Steps: Enter valid Jira key → Click Generate
Expected: Positive and Negative test cases displayed.
PS002 - Expand Steps
Steps: Click "View Steps"
Expected: Steps and Expected Result displayed.
PS003 - Export Test Cases
Steps: Generate tests → Click Export to Excel
Expected: CSV file downloaded.
PS004 - Copy Test Cases
Steps: Click Copy Test Cases
Expected: Test cases copied to clipboard.
PS005 - Toggle Theme
Steps: Click Toggle Theme
Expected: UI switches between Dark and Light mode.

Negative Scenarios

NS001 - Empty Jira Key
Steps: Click Generate without entering key
Expected: Validation error displayed.
NS002 - Invalid Jira Key
Steps: Enter invalid key like "ABC123"
Expected: Error message returned.
NS003 - API Failure
Steps: Simulate server failure
Expected: "Error loading test cases" shown.
NS004 - Large Response
Steps: Generate 50+ test cases
Expected: UI scroll works correctly.
NS005 - Export Without Data
Steps: Click Export before generating tests
Expected: No download or warning shown.
